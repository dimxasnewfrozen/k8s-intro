# YAML

yaml files contain the configuration data and resembles json.

see the examples folder for yaml syntax examples.

```
# Key Value

Fruit: Apple
```

```
# Array/Lists

Fruits:
    - Organge
    - Apple
```

```
# Dictionary/Map:

Banana:
    Calories: 105
```

```
# Key Value/Dictionary/Lists:

Fruits:
    -   Banana:
            Calories: 105
    -   Grape:
            Calories: 10
```

### Notes:
* Dictionary: unordered 
* List: ordered
* `#` denotes a comment
